# scalora-ai
Scalora AI is a simple web-based content generator designed to help creators quickly generate social media content ideas such as hooks, captions, and post ideas.

## Features

- Hook Generator
- Caption Generator (Upcoming Feature)
- Post Idea Generator (Upcoming Feature)

## Tech Stack

- HTML
- CSS
- JavaScript

## Current Functionality

The Hook Generator allows users to enter a niche and generate content hooks.  
The system:
- Accepts user input (niche)
- Validates empty input
- Dynamically generates hooks
- Replaces previous results with new output

## Architecture overview
User Input -> Javascript Logic -> Hook Generation -> UI Display
